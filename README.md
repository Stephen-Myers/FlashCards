# Flash Cards!

This is a personal project I built to help me study Japanese and other subjects more effectively.

I wanted an alternative to paid services like Quizlet and Anki, and I also found digital flashcards much more convenient than paper. So I decided to build my own app.

In the future, I plan to publish this app on iOS and continue adding new features.

---

## Features

Here are some of the features I’m most proud of:

### Deck Management
- Easily create and delete your own flashcard decks

### Custom Themes
- Personalize your experience with different themes

### Performance
- Fast processing speeds, even with:
  - 70+ double-sided flashcards
  - More complex study modes

## Study Modes

The app includes multiple study modes to fit different learning styles:

1. **Basic**
   - Standard yes/no flashcard review

2. **Full Mix**
   - Randomly shows either the front or back of cards

3. **Reverse**
   - Starts with the back side of cards

4. **Length-Based**
   - Study for a specified duration  
   - If left blank, cycles through the entire deck

5. **Type It In**
   - Users must type answers
   - Includes grading functionality

6. **Timed Mode**
   - Tracks average response time
   - Displays results at the end

7. **Combination Mode**
   - Designed for language learning (e.g., Japanese)
   - Dynamically combines inputs from cards
   - Creates new variations each session for deeper learning

8. **Image Upload**
   - Add images to flashcards
   - Useful for visual learners or hard-to-type content
